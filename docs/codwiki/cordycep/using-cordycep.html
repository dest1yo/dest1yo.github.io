使用Cordycep

Cordycep是一种利用游戏转储为我们加载游戏资源文件的工具，旧版本名为Parasyte。
配合Greyhound，我们能够绕过游戏反作弊，对游戏进行离线提取。
目前它只有命令行形式，但是使用起来其实很简单。

作者的Github: Scobalula
作者的交流频道: Discord服务器
赞助发电: https://ko-fi.com/scobalula

可以简单地认为

Cordycep可以运行一个无反作弊的游戏客户端，我们需要手动地加载资源文件。

下载​

  • Cordycep
    Cordycep的正常获取途径为：
    打赏作者，在Discord的赞助者频道获取下载地址。
  • 下载 Scylla

安装​

  1. 解压Cordycep压缩包，确保解压路径 不含中文。
  2. 把 Scylla 里的 Scylla_x64.dll 放入 Cordycep/Data/Deps 路径。
    (因为Cordycep需要Scylla库进行dump操作)
  3. Dump一份想提取的游戏 (如何Dump) (能正常使用则不需要重复操作)
  4. 编辑 Run游戏名.bat ，将引号内路径修改成自己的 游戏根目录
  5. 运行修改好的bat，等待软件自动加载游戏的基础文件
  6. 至此Cordycep环境准备完毕，接下来需要安装Greyhound
